Run
-----------
```

$ npm install --save-dev gulp

$ npm install

$ gulp
```


Requirements
-----------
* [NodeJS](http://nodejs.org/) v8.11.3 or latest installed.
* [npm](https://www.npmjs.com/) v5.6.0 or latest installed.
* [gulpJS](https://gulpjs.com/) installed.
